
# SISP Module M1: Orbital Geometry and Network Topology

## 1. Reference Frame and State Vectors
Calculations are performed in the Earth-Centered Inertial (ECI) coordinate system using an SGP4 propagator.
* **Position:** $\mathbf{r}_A(t)$ and $\mathbf{r}_B(t)$
* **Velocity:** $\mathbf{v}_A(t)$ and $\mathbf{v}_B(t)$
* **Orbital Radii:** $R_A = \|\mathbf{r}_A(t)\|$ and $R_B = \|\mathbf{r}_B(t)\|$

## 2. Line of Sight (Earth Blockage)
An exclusion sphere is defined as $R_{excl} = R_E + h_{min}$, where $R_E \approx 6371\text{ km}$ and $h_{min} \approx 100\text{ km}$ to avoid dense atmospheric attenuation.

The Earth central angle $\gamma(t)$ between two satellites is:
$$\gamma(t) = \arccos\left(\frac{\mathbf{r}_A(t) \cdot \mathbf{r}_B(t)}{R_A \cdot R_B}\right)$$
The maximum theoretical angle before Earth grazes the line of sight is:
$$\gamma_{max} = \arccos\left(\frac{R_{excl}}{R_A}\right) + \arccos\left(\frac{R_{excl}}{R_B}\right)$$
**Visibility Condition:** A clear Line of Sight (LoS) exists if $\gamma(t) < \gamma_{max}$.

## 3. Slant Range and Doppler Range Rate
The Slant Range vector is $\mathbf{d}(t) = \mathbf{r}_B(t) - \mathbf{r}_A(t)$, with distance $d(t) = \|\mathbf{d}(t)\|$.

To calculate the time-varying Doppler shift, we project the relative velocity onto the line-of-sight vector to find the **Range Rate**, $\dot{d}(t)$:
$$\dot{d}(t) = \frac{\mathbf{d}(t) \cdot (\mathbf{v}_B(t) - \mathbf{v}_A(t))}{d(t)}$$
The precise Doppler shift $\Delta f$ fed to the physical layer is:
$$\Delta f = f_c \cdot \frac{\dot{d}(t)}{c}$$

## 4. System Integration: 5-Second Snapshot
SISP mitigates Out-of-Sequence Measurements (OOSM) for its Hybrid Kalman/SVD algorithm by relying on the predictable visibility windows generated by this model. 
When a fault is detected, a satellite broadcasts a `CORRECTION_REQ`. The state machine initiates a strict **5-second collection timer**. Neighbors in the visibility window process the request and respond immediately. This allows the Weighted Median/Kalman algorithm to operate on a synchronous, time-aligned batch of spatial data, bypassing complex delayed-state propagation math.

import numpy as np
from skyfield.api import load, EarthSatellite

def calculate_orbital_visibility():
    ts = load.timescale()
    
    tle1_l1 = "1 45176U 20001A   23274.50000000  .00000000  00000-0  00000-0 0  9998"
    tle1_l2 = "2 45176  53.0000 180.0000 0001000   0.0000   0.0000 15.00000000    05"
    
    tle2_l1 = "1 45177U 20001B   23274.50000000  .00000000  00000-0  00000-0 0  9999"
    tle2_l2 = "2 45177  53.0000 185.0000 0001000   0.0000   0.0000 15.00000000    04"

    sat_A = EarthSatellite(tle1_l1, tle1_l2, 'Sat A', ts)
    sat_B = EarthSatellite(tle2_l1, tle2_l2, 'Sat B', ts)

    t = ts.utc(2026, 4, 19, 13, range(91)) # 90 minutes

    pos_A = sat_A.at(t).position.km
    pos_B = sat_B.at(t).position.km
    vel_A = sat_A.at(t).velocity.km_per_s
    vel_B = sat_B.at(t).velocity.km_per_s

    R_excl = 6371.0 + 100.0 # Earth + atm clearance
    f_c = 26e9
    c_km = 299792.458 

    print(f"{'Time(m)':<8} | {'LoS':<6} | {'Distance(km)':<15} | {'Doppler(kHz)':<15}")
    print("-" * 55)

    for i in range(len(t)):
        rA_vec, rB_vec = pos_A[:, i], pos_B[:, i]
        vA_vec, vB_vec = vel_A[:, i], vel_B[:, i]

        rA_mag, rB_mag = np.linalg.norm(rA_vec), np.linalg.norm(rB_vec)

        # Line of Sight
        dot_prod = np.dot(rA_vec, rB_vec) / (rA_mag * rB_mag)
        gamma = np.arccos(np.clip(dot_prod, -1.0, 1.0))
        gamma_max = np.arccos(R_excl / rA_mag) + np.arccos(R_excl / rB_mag)
        los_clear = gamma < gamma_max

        # Slant Range & Doppler
        d_vec = rB_vec - rA_vec
        d_mag = np.linalg.norm(d_vec)
        range_rate = np.dot(d_vec, vB_vec - vA_vec) / d_mag 
        doppler_khz = (f_c * (range_rate / c_km)) / 1000.0

        if i % 5 == 0 or not los_clear:
            status = "YES" if los_clear else "NO"
            print(f"{i:<8} | {status:<6} | {d_mag:<15.2f} | {doppler_khz:<15.2f}")

if __name__ == "__main__":
    calculate_orbital_visibility()
